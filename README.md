# Personal Portfolio Website

This is my personal portfolio website, built to showcase my projects, skills, and experience as a first-year Computing student at TU Dublin. The website highlights my work in Python, HTML, and CSS, and provides links to my GitHub repositories for each project.



---

## Features

- About Me section with personal introduction and education
- Skills section listing technical and soft skills
- Projects section showcasing Python and web projects
- Contact section with email and social links
- Responsive layout for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS3
- GitHub Pages for hosting
